# Soal

Nilai dari $\log_2 8 + \log_3 27 - \log_5 125$ adalah ...

# Pilihan

A. 0
B. 1
C. 2
D. 3
E. 4

# Kunci

D

# Pembahasan

$\log_2 8 = 3$ karena $2^3 = 8$
$\log_3 27 = 3$ karena $3^3 = 27$
$\log_5 125 = 3$ karena $5^3 = 125$

Maka: $3 + 3 - 3 = 3$. Jawaban **D**.

# Trap

Banyak siswa terjebak di opsi C karena salah hitung $\log_5 125$ sebagai 2.
Pastikan ingat: $5^3 = 125$, bukan $5^2$.
